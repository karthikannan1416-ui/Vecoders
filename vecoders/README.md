# VECoders Technical Club Website

Static club website demo built with HTML, CSS, and JavaScript.

## Features
- Animated opening screen with club logo.
- Login and registration system with separate roles: student, staff, volunteer, president, vice-president, secretary, vice-secretary.
- Role-based editing: authorized members can post news, manage gallery, and download registration data.
- Homepage news feed, videos, past events, gallery, event idea submission, and coordinator/head details.
- Excel download for registration data using SheetJS.

## Run locally
Open `index.html` in your browser.

## Notes
Data is stored in browser `localStorage` for demo purposes.
